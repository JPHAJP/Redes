Elementos de una red de area local.

Instrucciones PDF

Computadora

Dispositivo final que realiza tareas de procesamiento de datos, conecta a la red para acceder y compartir recursos.

Elemento Activo (IP propia): Si.

Uso: Acceder a internet

Repetidor

Amplifica la señal de red para extender su alcance, sin cambiar el contenido de los datos transmitidos.

Elemento Activo (IP propia): No.

Uso: Extender la cobertura de una red

HUB

Dispositivo que conecta varios segmentos de red, retransmitiendo los datos a todos los puertos, sin filtrado ni inteligencia de red.

Elemento Activo (IP propia): No.

Uso: Conectar múltiples computadoras en una pequeña red local

Switch

Conecta dispositivos en una red y usa direcciones MAC para enviar datos solo al dispositivo de destino específico.

Elemento Activo (IP propia): No.

Uso: Crear una red local eficiente, reduciendo colisiones de datos y mejorando el rendimiento.

Bridge

Conecta y filtra tráfico entre dos redes locales o segmentos de red, operando en la capa de enlace de datos (capa 2).

Elemento Activo (IP propia): No.

Uso: Conectar dos segmentos de red, mejorando el control y el filtrado de tráfico.

Router

Dispositivo que conecta diferentes redes y dirige el tráfico de datos entre ellas, operando en la capa de red y asignando direcciones IP.

Elemento Activo (IP propia): Si.

Uso: Conectar una red local a internet y enrutar el tráfico entre diferentes subredes.
