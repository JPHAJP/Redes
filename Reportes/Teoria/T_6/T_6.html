Estilos e inputs

Regresar a Teoría

Descripción

En esta actividad se explorará la implementación de estilos CSS para mejorar la presentación visual de una página web. CSS, o Cascading Style Sheets, permite a los desarrolladores controlar el diseño y apariencia de múltiples páginas web a la vez sin alterar el contenido HTML.Con ello se puede aplicar estilos como colores, fuentes, y espaciados, además de técnicas más avanzadas como la animación y los media para adaptar el diseño a diferentes dispositivos. Esto se logra mediante el uso de clases en donde con herencia se pueden obtener los estilos deseados para muchos contenidos.

Además, se muestran los distintos tipos de inputs en html para poder conocer su funcionamiento.
