Práctica 8

Opcional: Armado de cableado ethernet.

Regresar a Laboratorio

Descripción

En esta práctica tiene como objetivo aprender de manera práctica a manejar las conexiones de cable ethernet. Por ese motivo se armaron con conectores RJ45, cable ethernet CAT6 y pinzas ponchadoras un cableado ethernet tanto directo como cruzado.

Instrucciones PDF

Existen 2 configuraciones de cable de red:

Directo

Este tipo de cables se utiliza para conectar dispositivos diferentes como una PC o un switch o un router. En esta configuración, los extremos están conectados de la misma manera. Los estándares mas comunes son T568A y T568B.

Conexiones físicas más utilizadas:
o Switch – Router
o Switch – PC

Cruzado

Esta configuración se usa para conectar dispositivos similares como 2 PC. En un extremo tiene la configuración T568A y el otro T568B.

Conexiones físicas más utilizadas:
o Switch – Switch
o Router – Router
o Router – PC
o PC – PC


Probar cables

Para probar un cable de red se requiere un probador de cables ethernet donde un extremo se conecta al puerto de transmisión (TX) y el otro extremo al puerto de recepción (RX) y si existe una buena conexión las luces deben encenderse en orden.


En caso de no contar con el tester, se puede probar la conexión de un cable directo conectando un extremo a una PC y el otro extremo a un switch o router. Ambos deben estar conectados a la misma red.

Se debe abrir la terminal del PC y utilizando el comando “ping” más la dirección IP del otro dispositivo, se mandan una serie de paquetes, si se reciben respuestas sin pérdida de paquetes, el cable está funcionando correctamente.

Para probar un cable cruzado, se conecta cada extremo del cable a una PC diferente que estén conectadas a la misma subred y repitiendo el comando “ping” mas la IP del otro dispositivo, se mandan una serie de paquetes, si se reciben respuestas sin pérdida de paquetes, el cable está funcionando correctamente.

Armados:

Para armar un cable directo se necesita un cable ethernet (CAT6), conectores RJ45 y una pinza ponchadora. Se debe cortar el cable a la longitud deseada y se debe pelar el cable para exponer los 8 cables internos. Se deben acomodar los cables en el orden deseado, ya sea T568A o T568B, y se debe cortar el exceso de cable. Se debe insertar el cable en el conector RJ45 y se debe utilizar la pinza ponchadora para asegurar el cable al conector. Se debe repetir el proceso en el otro extremo del cable.
