Práctica 9

Opcional: Configuración de red local.

Regresar a Laboratorio

Descripción

En esta práctica se busca poder conocer e identificar los pasos para poder configurar una red local y accesar a los recursos de uno de los equipos en la red.

Instrucciones PDF
Ejemplo en red local con conexión inalámbrica

El video muestra la creación de un servidor web para ser visualizado en la red de la universidad mediante dispositivos inalámbricos. En este paso resulto complicado enconrar computadoras que tuvieran compunicación ya que la red se encuentra segmentada (una conexión entre computadoras cableadas e inalámbricas en la red universitaria fué imposible).

Ejemplo en red local con conexión alámbrica

El video muestra la creación de un servidor web para ser visualizado en la red de la universidad mediante dispositivos conectados mediante cableado a la red.

Ejemplo en red local con conexión alámbrica computadora a computadora

El video muestra la creación de un servidor web para ser visualizado; se realiza una configuración IP del servidor web y del cliente ya que es una conexión punto a punto y se establece una IP manualmente (para el servidor web 192.168.0.1 y para el cliente 192.168.0.2).
